Removal

NOTE: Installation of a new suction accumulator is not required when repairing the air conditioning system, except when there is physical evidence of system contamination from a failed A/C compressor or damage to the suction accumulator.

  1. Recover the refrigerant. For additional information, refer to Section 412-00 .
  1. Disconnect the evaporator core orifice access fitting.
    • Discard the O-ring seal.
  1. Inspect the evaporator core orifice for damage before attempting to remove it from the line.
  1. If the evaporator core orifice is intact, engage the special tool. Hold the T-handle stationary while rotating the tool body to remove the evaporator core orifice.
  1. If the evaporator core orifice is broken, screw the end of the special tool into the broken orifice. Hold the T-handle stationary while rotating the tool body to remove the evaporator core orifice.

Installation

  1. Lubricate the evaporator core orifice O-rings in clean PAG oil and install the evaporator core orifice using the special tool.
  1. Connect the evaporator core orifice access fitting.
    • Install a new O-ring seal.
  1. Lubricate the refrigerant system with the correct amount of clean PAG oil. For additional information, refer to Section 412-00 .
  1. Evacuate, leak test and charge the refrigerant system. For additional information, refer to Section 412-00 .
