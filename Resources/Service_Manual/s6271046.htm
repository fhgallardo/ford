SECTION 307-01: Automatic Transmission — 4R70E/4R75E
2006 F-150/Mark LT Workshop Manual
IN-VEHICLE REPAIR
Procedure revision date: 06/29/2005

Overdrive Servo


Special Tool(s)
Remover/Installer, Servo Piston
307-251 (T92P-70023-A)

Removal

  1. With the vehicle in NEUTRAL, position it on a hoist. For additional information, refer to Section 100-02 .
  1. Remove the main control valve body. For additional information, refer to Main Control Valve Body in this section.

  1. NOTE: If the tool is not available, extreme care must be taken. Spring pressure will force overdrive servo piston assembly out of case. Case bore damage may result from trying to pry on overdrive servo internal retaining ring.

    Using the special tool, compress the servo spring to remove the overdrive servo retaining ring.
  1. Remove the overdrive servo piston and return spring.

Installation

  1. Install the overdrive servo piston assembly.
    1. Install the overdrive servo piston return spring.
    1. Install the overdrive servo piston assembly.
  1. Use the special tool to install the overdrive servo piston retaining ring.
