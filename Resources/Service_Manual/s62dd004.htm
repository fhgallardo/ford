SECTION 413-13: Parking Aid
2006 F-150/Mark LT Workshop Manual
GENERAL PROCEDURES
Procedure revision date: 06/29/2005

Azimuth System Check

Maximum Detectable Object Locations

  1. NOTE: The object used in this system check can be fabricated from 7.62 cm (3 in) diameter pipe 99.06 cm (39 in) length.

    NOTE: The following system check should be carried out with the vehicle on a level surface.

    NOTE: The parking aid system will default to on when the ignition key is cycled from off to on.

    Turn the ignition switch to the ON position, engine off.
  1. Set the parking brake on.
  1. Place the transmission in REVERSE.
  1. Verify that the parking aid system detects the specified object when placed in the five specified locations (P1, P2, P3, P4, P5).
