SECTION 418-00: Module Communications Network
2006 F-150/Mark LT Workshop Manual
DESCRIPTION AND OPERATION
Procedure revision date: 06/29/2005

Communications Network

Vehicle Security Module

The vehicle security module uses the universal asynchronous receiver transmitter (UART)-based protocol (UBP) communications network. The vehicle security module controls the remote keyless entry (RKE) and power door lock functions for the vehicle.

For additional RKE information, refer to Section 501-14 .

For additional vehicle security module information, refer to Section 419-10 .

Powertrain Control Module (PCM)

The PCM uses the controller area network (CAN) to communicate with the diagnostic tool. The PCM uses an internal termination resistor for normal interface with the CAN. Damage or loss of this termination resistor results in intermittent operation and or failure of the entire CAN. The PCM controls the engine for better fuel economy, emissions control, failure mode detection and storage as well as all 4-wheel drive and speed control functions.

For additional 4-wheel drive information, refer to Section 308-07A .

For additional speed control information, refer to Section 310-03 .

Refer to the Powertrain Control/Emissions Diagnosis (PC/ED) manual for diagnosis and testing of the PCM.

Restraint Control Module (RCM)

The RCM uses the international standards organization (ISO) 9141 communications network to communicate with the diagnostic tool. The RCM controls the deployment of the air bags based on sensor input.

For additional RCM information, refer to Section 501-20B .

Parking Aid Module

The parking aid module uses the ISO 9141 communications network to communicate with the diagnostic tool. The module controls sensors in the rear bumper that detect close objects when the vehicle is in reverse.

For additional parking aid module information, refer to Section 413-13 .

Instrument Cluster

The instrument cluster uses both the CAN and the UBP communications networks to communicate with network modules and the diagnostic tool. It is an analog face instrument cluster with a digital odometer. The instrument cluster displays a variety of vehicle systems information.

For additional instrument cluster information, refer to Section 413-01 .

Electronic Automatic Temperature Control (EATC) Module

The EATC module uses the UBP communications network. The EATC module controls automatic climate functions that maintain the vehicle interior temperature at a constant setting.

For additional climate control assembly module information, refer to Section 412-00 .

Anti-Lock Brake System (ABS) Module

The ABS module uses the CAN to communicate with the diagnostic tool. The ABS module controls the brake pressure to the 4 wheels to keep the vehicle under control while braking.

For additional ABS module information, refer to Section 206-09 .
