Disassembly

 CAUTION: Discard all seals after removing them.

  1. Remove the transfer case. For additional information, refer to Transfer Case — Electronic Shift in this section.

  1.  WARNING: Wear protective eyewear whenever using compressed air.

    Clean the transfer case with solvent and dry with compressed air.

  1.  WARNING: Make sure the special tool lock pin is secure.

    Using the special tool, secure the transfer case to the bench.
  1. If not done previously, remove the drain plug and drain the fluid.
  1. Review the transfer case symptom chart before and during the disassembly procedure.

    Electronic Shift Transfer Case Symptom Chart
    Condition Possible Source(s) Action

    · Fluid contaminated (plastic/metallic)

    • Seal failure (items 23, 27, 49, 56).
    • Oil pump assembly (item 8).

    · INSPECT seals for signs of ingress or egress. If metal and plastic contamination is noted when the fluid is drained, OPEN transfer case and INSPECT internal components for wear. INSTALL new components as necessary, including the transfer case pump. VERIFY correct operation of all components prior to reassembly. All contamination should be cleaned from the components. All bearings should be inspected for smooth operation. If issue not resolved, CONTACT Service Hotline for direction.

    · Jumps out of 4WD (even when shift motor removed)

    • Lockup fork (item 65).
    • Reduction shift fork assembly (item 66).
    • Shift cam assembly (item 62).
    • Lockup hub assembly (item 12).
    • Oil pump assembly (item 8).

    · INSPECT internal shift system for damage or wear. Shift forks and shift fork pads should be inspected for wear or damage. If signs of heat generation are noted, INSTALL a new pump assembly and INSTALL pump filter. INSPECT slip yokes and bushings for unusual wear. If issue not resolved, CONTACT Service Hotline for direction.

    · 4HI will not engage (when shift cam is positioned manually in 4H with shift motor removed)

    • Transfer case shift components:
      • Lockup fork (item 65).
      • Lockup hub assembly (item 12).
      • Shift cam assembly (item 62).
      • Reduction shift fork assembly (item 66).
    • Oil pump assembly (item 8).

    · INSPECT internal shift components for wear. Shift forks and shift fork pads should be inspected for wear or damage. If signs of heat generation are noted, INSTALL a new pump assembly and INSTALL pump filter. INSPECT slip yokes and bushings for unusual wear. If issue not resolved, CONTACT Service Hotline for direction.

    · 4LO will not engage (when shift cam is positioned manually in 4L with shifter linkage removed)

    • Lockup fork (item 65).
    • Reduction shift fork assembly (item 66).
    • Reduction hub assembly (item 7).
    • Shift cam assembly (item 62).
    • Lockup hub assembly (item 12).
    • Oil pump assembly (item 8).

    · INSPECT internal shift components for wear. Shift forks and shift fork pads should be inspected for wear or damage. If signs of heat generation are noted, INSTALL a new pump assembly and INSTALL pump filter. INSPECT slip yokes and bushings for unusual wear. If issue not resolved, CONTACT Service Hotline for direction.

    · 4WD will not disengage (when shift cam is positioned manually in 2H with shifter motor removed)

    • Transfer case shift components:
      • Lockup fork (item 64).
      • Lockup Hub assembly (item 12).
      • Shift cam assembly (item 62).
      • Reduction shift fork assembly (item 66).
    • Oil pump assembly (item 8).

    · INSPECT internal shift components for wear. Shift forks and shift fork pads should be inspected for wear or damage. If signs of heat generation are noted, INSTALL a new pump assembly and INSTALL pump filter. INSPECT slip yokes and bushings for unusual wear. If issue not resolved, CONTACT Service Hotline for direction.

    · Not able to manually shift transfer case

    • Transfer case shift components:
      • Lockup fork (item 65).
      • Lockup hub assembly (item 12).
      • Shift cam assembly (item 62).
      • Reduction shift fork assembly (item 66).
    • Oil pump assembly (item 8).

    · INSPECT internal shift components for wear. Shift forks and shift fork pads should be inspected for wear or damage. If signs of heat generation are noted, INSTALL a new pump assembly and install pump filter. INSPECT slip yokes and bushings for unusual wear. If issue not resolved, CONTACT Service Hotline for direction.

    · NVH

    • Loose driveshafts.
    • Loose screws or bolts.
    • Drive chain (item 20).
    • Front planet carrier (item 4).
    • Shift components.

    · INSPECT system for loose screws or bolts. TIGHTEN as required. DRAIN fluid and INSPECT for contamination. If heavy metallic or plastic contamination noted, REMOVE transfer case and INSPECT for source of contamination. INSTALL a new components as necessary. If signs of heat generation are noted, INSTALL a new pump assembly and INSTALL pump filter. INSPECT driveshaft slip yoke and INSTALL new as required. If issue not resolved, CONTACT Service Hotline for direction.

    · Uncommanded shift

    • Transfer case shift components:
      • Lockup fork (item 65).
      • Lockup hub assembly (item 12).
      • Shift cam assembly (item 62).
      • Reduction shift fork assembly (item 66).
    • Shift fork pads worn:
      • Reduction hub assembly (item 7).

    · INSPECT internal shift components for wear. Shift forks and shift fork pads should be inspected for wear or damage. If signs of heat generation are noted, INSTALL a new pump assembly and INSTALL pump filter. INSPECT slip yokes and bushings for unusual wear. If issue not resolved, CONTACT Service Hotline for direction.

    · Rear output leak

    • Yoke to flange seal (item 56).
    • Slip yoke damage or degradation.
    • Internal or external contamination.

    · INSPECT yoke for damage or excessive wear. If wear noted on yoke, INSPECT transfer case rear output bushing for damage. INSPECT transfer case seal for damage or wear. INSPECT system for contamination. INSTALL a new slip yoke, transfer case bushing and/or seal as required. If issue not resolved, CONTACT Service Hotline for direction.

    · Transfer case overfull

    • Input flange seal (item 23).

    · CHECK transmission for low fluid condition. If transmission fluid level OK, DRAIN and REFILL transfer case to correct level. If transmission fluid level low, INSPECT the input flange seal for damage. If issue not resolved, CONTACT Service Hotline for direction.

    · T-case is in a range/mode other than indicated by motor contact plates (uncommanded shift/jumps out of gear)

    • Transfer case shift components:
      • Lockup fork (item 65).
      • Lockup hub assembly (item 12).
      • Shift cam assembly (item 62).
      • Reduction shift fork assembly (item 66).
    • Oil pump assembly (item 8).

    · INSPECT internal shift components for wear. Shift forks and shift fork pads should be inspected for wear or damage. If signs of heat generation are noted, INSTALL a new pump assembly and INSTALL pump filter. INSPECT slip yokes and bushings for unusual wear. If issue not resolved, CONTACT Service Hotline for direction.


  1. Using a suitable tool, remove the oil seal.
  1. Remove the coil wire and the ground wire from the electric shift motor electrical connector.
    • Remove the wire connector spacer.
    • Remove the coil wire and pin.
    • Remove the ground wire and pin.
      • Use an electrical connector pin extractor tool.
  1. Remove the transfer case shift motor.
    1. Remove the bolts.
    1. Remove the transfer case shift motor.
  1. Remove the extension housing.
    1. Remove the 6 extension housing bolts.
    1. Remove the extension housing.
  1. Remove the rear output shaft snap ring.
    • Lift up on the output shaft while removing the snap ring.

  1. NOTE: The bolts are self-tapping, and it is normal to find metal shavings while removing the cover.

    Remove the bolts.

  1. NOTE: Use pry bosses to separate the cover from the case.

    Separate the transfer case covers.
  1. Remove the coil.
    1. Remove the 3 coil-to-cover nuts.
    1. Remove the coil.
  1. Using the special tools, remove the rear output shaft bearing.
  1. Using the special tools, remove the front output shaft bearing.
  1. Remove the shift collar hub snap ring.
  1. Remove the coil housing.
  1. Remove the shift fork.
    1. Remove the lockup hub.
    1. Remove the shift fork and spring.

  1. NOTE: Remove the lockup hub assembly as a single unit. If disassembly is required, proceed with the following.

    Disassemble the lockup hub assembly.
    1. Remove the lockup hub snap ring.
    1. Separate the lockup collar.
    1. Separate the 2 lockup springs.
    1. Separate the lockup hub.
    1. Remove the armature.
  1. Remove the spacer.
  1. Remove the drive chain and sprockets as an assembly.
  1. Remove the magnet from its slot in the case.

  1. NOTE: The pump assembly is not repairable.

    Remove the oil pump and the output shaft as an assembly.
    • Remove the oil pump from the output shaft.
  1. Remove the shift rail.
  1. Remove the shift fork and the reduction hub.
    1. Remove the shift fork.
    1. Remove the reduction hub.

  1.  WARNING: Do not disassemble shift cam assembly.

    Remove the electric shift cam assembly.
  1. Using the special tool, remove the input seal.

  1.  CAUTION: Do not let the front planet carrier fall while removing the snap ring.

    Remove the front planet carrier.
    1. Remove the front planet carrier snap ring.
    1. Remove the front planet carrier.
  1. If necessary, using the special tools, remove the bushing from the front planet.
  1. Using a drift and a hammer, remove the needle bearing, if necessary.
  1. Remove the ring gear.
    1. Remove the ring gear snap ring.
    1. Remove the ring gear.
  1. Remove the front planet carrier bearing snap ring. Then, using the special tools, remove the front planet carrier bearing.

  1.  CAUTION: Do not let the front output shaft fall while removing the snap ring.

    Remove the front output shaft.
    1. Remove the front output shaft snap ring.
    1. Remove the front output shaft.
  1. Using the special tools, remove the front output shaft bearing.
  1. Using the special tools, remove the front yoke to flange seal.
  1. Using a suitable pry tool, remove the electric shift motor seal.
  1. Remove the dust shield, if necessary.

Assembly

  1.  CAUTION: Do not crush the seal.

    Using the special tool, install the electric shift motor seal.
  1. Using a suitable press and the special tools, install the front output shaft bearing.
  1. Using the special tool, install the front yoke to flange seal.
  1. Install the dust shield, if removed.
  1. Install the front output shaft.
    1. Position the front output shaft.
    1. Install the front output shaft snap ring.
  1. Using a press and a suitable bearing installer, install the front planet carrier bearing. Install the front planet carrier bearing snap ring.
  1. Install the ring gear.
    • Position the ring gear.
    • Install the ring gear snap ring.

  1.  CAUTION: Do not crush the needle bearings.

    If removed, using a suitable press and the special tool, install the needle bearing and bushing into the front planet carrier.
  1. Install the front planet carrier.
    1. Position the front planet carrier.
    1. Install the front planet carrier snap ring.
  1. Using the special tools, install the input seal.
  1. Install the electric shift cam assembly.
  1. Install the shift fork and the reduction hub.
  1. Install the shift rail.
  1. Install the oil pump and the output shaft.
  1. Install the magnet into its slot in the case.
  1. Install the drive chain and sprocket assembly.
  1. Install the spacer.

  1. NOTE: If the lockup hub assembly was disassembled, carry out the following steps.

    Assemble the lockup hub assembly.
    1. Position the lockup collar.
    1. Install the lockup collar springs.
    1. Install the lockup hub.
    1. Install the armature.
    1. Install the lockup hub snap ring.
  1. Install the lockup hub and the shift fork and spring.
  1. Install the coil housing.
  1. Install the snap ring.
  1. Using the special tool, install the rear output shaft bearing.
  1. Using the special tool, install the front output shaft bearing.
  1. Install the coil.
    1. Position the coil.
    1. Install the nuts.
      • Tighten to 10 Nm (89 lb-in).

  1.  CAUTION: If too much silicone rubber is used when sealing the cover to the case, the excess sealant can plug the fluid filter and cause case failure.

    Coat the mating surfaces of the case and cover with a small bead of sealant.
  1. Position the 2 transfer case halves and tighten the bolts according to sequence shown.
    • Tighten to 24 Nm (18 lb-ft).
  1. Install the rear output shaft snap ring.
    • Lift up on the output shaft while installing the snap ring.
  1. Apply a small bead of sealant to the transfer case cover and extension housing mating surfaces.
  1. Install the extension housing.
    1. Position the brown clutch coil wire through its hole in the extension housing.
    1. Position the extension housing.
    1. Install the extension housing bolts.
      • Tighten to 24 Nm (18 lb-ft).
  1. Using the special tool, install the oil seal, if removed.

  1. NOTE: Tighten the 3 motor bolts first, then the bracket bolt.

    Install the transfer case shift motor.
    • Tighten to 10 Nm (89 lb-in).
  1. Install the coil wire into the shift motor electrical connector.
    • Install the coil wire and pin.
    • Install the ground wire and pin.
    • Install the wire connector spacer.
  1. Remove the transfer case from the special tool.
  1. Install the transfer case in the vehicle. For additional information, refer to Transfer Case in this section.
