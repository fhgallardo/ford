REMOVAL AND INSTALLATION
Procedure revision date: 06/29/2005

Window Control Switch — Rear

NOTE: LH shown, RH similar.


Item Part Number Description
1 — Window control switch bezel
2 — Window control switch electrical connectors
3 — Window control switch

Removal and Installation

  1. Remove the window control switch bezel.
  1. Disconnect the window control switch electrical connectors.
  1. Remove the window control switch.
  1. To install, reverse the removal procedure.
