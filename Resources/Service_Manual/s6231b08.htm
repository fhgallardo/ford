Removal

  1. Remove the crankshaft pulley. For additional information, refer to Crankshaft Pulley in this section.
  1. Using the special tool, remove and discard the crankshaft front seal.

Installation

  1. Lubricate the engine front cover and the crankshaft front seal inner lip with clean engine oil.
  1. Using the special tools, install a new crankshaft front seal.
  1. Install the crankshaft pulley. For additional information, refer to Crankshaft Pulley in this section.
