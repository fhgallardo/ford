Removal

  1. Remove the flexplate. For additional information, refer to Flexplate in this section.
  1. Remove the engine rear cover plate.
  1. Using the special tools, remove and discard the crankshaft oil slinger.
  1. Using the special tools, remove and discard the crankshaft rear seal.

Installation

  1. NOTE: Lubricate the crankshaft rear seal with clean engine oil prior to installation.

    Using the special tools, install a new crankshaft rear seal.

  1. NOTE: Lubricate the crankshaft oil slinger with clean engine oil prior to installation.

    Using the special tools, install a new crankshaft oil slinger.
  1. Install the engine rear cover plate.
  1. Install the flexplate. For additional information, refer to Flexplate in this section.
