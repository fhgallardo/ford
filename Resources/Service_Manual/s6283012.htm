  1. Install the countershaft.
    • Do not install the countershaft front or rear bearings at this time.
  1. Position the input shaft into the transmission.

  1. NOTE: Make sure the 4th gear synchronizer blocking ring and the bearings are in place.

    Install the output shaft and mate the input shaft and output shaft together.
  1. Install the center countershaft bearing.
  1. Install the output shaft center bearing.

  1. NOTE: Install the rear bearing retainer with the arrow pointing to the top of the case.

    Install the rear bearing retainer and the 6 rear bearing retainer bolts.
    • Tighten to 22 Nm (16 lb-ft).

NOTE: If any related parts (such as the output shaft or bearing) have been installed new, carry-out the shim thickness selective procedure.

  1. Using the special tool, install the front cover oil seal.
  1. Position the transmission with the bellhousing facing up.
  1. Install the front countershaft bearing.
  1. Install the input shaft front bearing race.

  1. NOTE: A digital caliper is shown being used for the depth measurement. A dial indicator with depth capability or a depth micrometer can also be used.

    Measure the input shaft bearing race height.
    • Record this measurement on line 2 of the selective shim equation chart.
  1. Measure the front bearing retainer depth.
    • Record this measurement on line 1 of the selective shim equation chart.
  1. Calculate the measurements.

  1. Measure and record the countershaft bearing depth.
  1. Select the correct countershaft shim.
    • Select the appropriate spacer to achieve the correct end play. Countershaft end play is 0.15-0.25 mm (0.0059-0.0098 in).
    • Countershaft selection shims:
      • 3.0 mm (0.118 in)
      • 3.1 mm (0.122 in)
      • 3.2 mm (0.125 in)
      • 3.3 mm (0.129 in)
      • 3.4 mm (0.134 in)
      • 3.5 mm (0.138 in)
      • 3.6 mm (0.142 in)
      • 3.7 mm (0.146 in)

  1. NOTE: Apply a light coat of fluid or grease to the countershaft spacer, shim and fluid baffle.

    Install the bearing shim, fluid baffle and countershaft spacer onto the front bearing retainer.

  1. NOTE: Apply a small amount of silicone sealer to the bolt threads.

    NOTE: Apply a light coat of fluid to the seal lips. Be careful not to damage the seal when installing the retainer.

    Install the front bearing retainer.
    • Tighten to 22 Nm (16 lb-ft).

    1. NOTE: Apply a small amount of silicone sealer to the bolt threads.

      Install the fluid passage and the fluid passage bolt.
      • Tighten to 9 Nm (80 lb-in).
    1. Install the output shaft 5th speed gear.
    1. Install the following components:
      • Locating ball
      • Inner race
      • 5th speed gear split bearing
    1. Install the countershaft 5th speed gear.
    1. Position the counter lever shaft and plate, then install the counter lever lock plate bolt.
      • Tighten to 9 Nm (80 lb-in).
    1. Install the 5th/reverse shift lever, the washer and the 5th/reverse shift retaining ring.
    1. Install the countershaft 5th gear synchronizer blocking ring.

    1. NOTE: The reference mark on the synchronizer sleeve must face toward the reverse gear.

      Install the 5th/reverse shift rail, the shift fork and the 5th/reverse synchronizer assembly.
    1. Install the 5th/reverse shift rail bolt.
      • Tighten to 25 Nm (18 lb-ft).
    1. Install the countershaft split washer.
    1. Check clutch hub end play.
      • If clutch hub end play is not within specification, select and install the correct split washer.
      • Split washer selection:
        • 3.0 mm (0.118 in)
        • 3.1 mm (0.122 in)
        • 3.2 mm (0.125 in)
        • 3.3 mm (0.129 in)
        • 3.4 mm (0.133 in)
        • 3.05 mm (0.120 in)
        • 3.15 mm (0.124 in)
        • 3.25 mm (0.127 in)
        • 3.35 mm (0.131 in)
    1. Install the reverse gear thrust washer.
    1. Install the following:
      1. Reverse synchronizer blocking ring
      1. Two countershaft reverse gear bearings
      1. Countershaft reverse gear
    1. Measure the countershaft reverse gear end play.
      • Install the thrust washer and hold it against the shoulder on the countershaft while checking end play.
      • Determine the correct thrust washer to obtain correct end play.
      • Thrust washer selection:
        • 7.35 mm (0.289 in)
        • 7.45 mm (0.293 in)
        • 7.55 mm (0.297 in)
        • 7.65 mm (0.301 in)
        • 7.75 mm (0.305 in)
        • 7.85 mm (0.309 in)
    1. Install the output shaft spacer.
    1. Install the output shaft reverse gear.
    1. Install the reverse idler gear and bushing.
    1. Apply Threadlock and Sealer to the reverse idler shaft bolt, install the bolt.
      • Tighten to 98 Nm (72 lb-ft).
    1. Position the output shaft rear bearing on the output shaft. Using the old output shaft locknut, install the output shaft rear bearing. Discard the nut.
    1. Install the thrust washer and countershaft rear bearing.
    1. Lock the transmission into 1st and 3rd gear.
    1. Install a new output shaft locknut.
      • Tighten to 245 Nm (181 lb-ft).
    1. Install a new countershaft locknut.
      • Tighten to 162 Nm (119 lb-ft).
    1. Using the special tool, stake the output shaft locknut and the countershaft locknut.
    1. Position the locating ball on the shaft, install the output speed (OSS) tone ring, then install the snap ring.
    1. Apply a small bead of silicone rubber to the extension housing mating surface.
    1. Install the extension housing and the extension housing bolts.
      • Tighten to 39 Nm (29 lb-ft).
    1. Using the special tool, install a new extension housing seal.
    1. Place the synchronizer sleeves in NEUTRAL.
    1. Install the shift control housing seal.
    1. Install the case cover and the case cover bolts.
      • Tighten to 19 Nm (14 lb-ft).
    1. Install the clutch slave cylinder.
      • Tighten to 20 Nm (15 lb-ft).
