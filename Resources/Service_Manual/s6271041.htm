Removal

  1. With the vehicle in NEUTRAL, position it on a hoist. For additional information, refer to Section 100-02 .

  1. NOTE: The output shaft and driveshaft are a balanced assembly.

    Mark the driveshaft flange and the rear pinion flange for correct alignment during assembly.
  1. Remove the 4 driveshaft bolts.
  1. Slide the driveshaft back enough to index-mark the driveshaft, the extension housing and the output shaft. Separate the driveshaft from the transmission.
  1. Using the special tools, remove the extension housing seal.

Installation

  1. NOTE: When installing the new seal, make sure that the drain hole in the seal is in the 6 o'clock position.

    Using the special tool, install a new extension housing seal.

  1. NOTE: The output shaft and the driveshaft are a balanced assembly.

    Install the driveshaft.
    • Align all the index marks and position the driveshaft on the transmission.
    • Install the driveshaft.

  1. NOTE: To maintain driveshaft balance, install the driveshaft yoke and flange in the original position that was marked during removal.

    Install the rear driveshaft and the 4 bolts.
    • Tighten to 103 Nm (76 lb-ft).
  1. Check the level of the transmission fluid, add clean automatic transmission fluid as necessary.
