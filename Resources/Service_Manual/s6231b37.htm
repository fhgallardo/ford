  1.  CAUTION: Clean the engine support insulator-to-frame mating surfaces of any dirt or foreign material prior to engine installation.

    NOTE: Position the engine assembly forward to connect the RH heated exhaust gas oxygen sensor (HO2S) electrical connector and the transmission harness retainers at the rear of the cylinder head.

    Using a suitable floor crane, position the engine assembly into the vehicle.
  1. Apply threadlock to the bolt threads and install the LH engine support insulator bolt.
    • Tighten to 350 Nm (258 lb-ft).

  1.  CAUTION: Make sure the RH engine support insulator mating surfaces and the washer mating surface are free of foreign material and corrosion before installation. Install a new washer.

    Apply threadlock to the bolt threads and install the RH engine support insulator washer and nuts.
    • Tighten to 250 Nm (184 lb-ft).
  1. Install the 6 transmission-to-engine bolts.
    • Tighten to 60 Nm (44 lb-ft).
  1. Connect the shift cable and install the shift cable bracket and the bolts.
    • Tighten to 25 Nm (18 lb-ft).
  1. Connect the LH heated exhaust gas oxygen sensor (HO2S) electrical connector and attach the electrical connector retainer.

  1. NOTE: Lubricate the O-ring seals with clean transmission fluid.

    Position the transmission fluid filler tube and install the bolt.
    • Tighten to 12 Nm (9 lb-ft).
  1. Install the 4 exhaust manifold-to-catalytic converter nuts.
    • Tighten to 40 Nm (30 lb-ft).
  1. Install the torque converter-to-flexplate nuts.
    • Tighten to 35 Nm (26 lb-ft).
  1. Install the cylinder block opening cover.
  1. Install the flexplate inspection cover.
    • Tighten to 34 Nm (25 lb-ft).
  1. Install a new oil filter.
  1. Install the drain plug.
    • Tighten to 14 Nm (10 lb-ft).
  1. Attach the starter electrical harness support to the cylinder block.
    • Tighten to 10 Nm (89 lb-in).
  1. If equipped, connect the block heater electrical connector.
  1. Position the A/C compressor and install the bolts.
    • Tighten to 25 Nm (18 lb-ft).
  1. Install the starter. For additional information, refer to Section 303-06 .
  1. Position the power steering reservoir and install the 2 lower bolts.
    • Tighten to 23 Nm (17 lb-ft).
  1. Position the power steering pump and install the bolts.
    • Tighten to 25 Nm (18 lb-ft).
  1. Using the special tool, install a new O-ring seal on the pressure line fitting.
  1. Connect the power steering pressure tube.
    • Tighten to 65 Nm (48 lb-ft).
  1. Position the power steering pressure tube support bracket and install the nut.
    • Tighten to 12 Nm (9 lb-ft).

  1.  CAUTION: If the pulley has been removed and installed twice, install a new power steering pump pulley.

    Using the special tool, install the power steering pump pulley.
    • Inspect the pulley for paint marks in the web area near the hub. If there are 2 paint marks, install a new pulley. If there is one paint mark or none at all, use a pencil to mark the web area of the pulley near the hub.
  1. If equipped, install the transmission cooler tube support bracket and nut.
    • Tighten to 10 Nm (89 lb-in).
  1. Connect the degas bottle coolant hose.
  1. Connect the A/C compressor electrical connector.
  1. Connect the crankcase position (CKP) sensor electrical connector and the harness retainer.
  1. Position the A/C manifold and tube assembly support bracket and install the nut.
    • Tighten to 10 Nm (89 lb-in).
  1. Remove the special tool.
  1. Install the upper transmission-to-engine bolts
    • Tighten to 60 Nm (44 lb-ft).
  1. Connect the heater coolant hose.
  1. Install the ground strap and the bolt and connect the electrical connector.
    • Tighten to 10 Nm (89 lb-in).
  1. Install the cowl panel extension.
  1. Attach the wiring harness and the windshield washer hose to the cowl panel extension.
  1. Install the cowl panel grille. For additional information, refer to Section 501-02 .
  1. Install the powertrain control module (PCM) and the support bracket. For additional information, refer to Section 303-14 .
  1. Install the intake manifold. For additional information, refer to Intake Manifold in this section.
  1. Install the radiator. For additional information, refer to Section 303-03 .
  1. Install the hood and the 4 bolts.
    • Tighten to 30 Nm (22 lb-ft).
  1. Fill the engine with clean engine oil.
  1. Fill and bleed the engine cooling system. For additional information, refer to Section 303-03 .
