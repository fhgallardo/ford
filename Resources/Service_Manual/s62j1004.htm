SECTION 419-01: Anti-Theft
2006 F-150/Mark LT Workshop Manual
DIAGNOSIS AND TESTING
Procedure revision date: 06/29/2005

Passive Anti-Theft System (PATS) — Inspection and Verification

  1. Verify the customer concern.
  1. Visually inspect for obvious signs of mechanical or electrical damage.


Visual Inspection Chart
Mechanical Electrical
  • Large metallic objects, a second PATS key on the same key ring as the PATS key, or electronic devices on the key chain that can be used to purchase gasoline or similar items
  • Ignition lock cylinder
  • PATS key
  • Use of a non-PATS key
  • Central junction box (CJB) fuse(s):
    • 5 (7.5A)
    • 28 (5A)
  • Circuitry
  • Connector(s)
  • PATS transceiver
  • Key switch
  • Light sensor
  • Powertrain control module (PCM)

  1. If an obvious cause for an observed or reported concern is found, correct the cause (if possible) before proceeding to the next step.
  1. If the cause is not visually evident, connect the diagnostic tool to the data link connector (DLC) and select the vehicle to be tested from the diagnostic tool menu. If the diagnostic tool does not communicate with the vehicle:
    • check that the program card is correctly installed.
    • check the connections to the vehicle.
    • check the key position.
  1. If the diagnostic tool still does not communicate with the vehicle, refer to the diagnostic tool operating manual.
  1. Carry out the diagnostic tool data link test. If the diagnostic tool responds with:
    • CAN, UBP or ISO circuits fault; all electronic control units no response/not equipped, refer to Section 419-10 .
    • No response/not equipped for the PCM, refer to Section 418-00 .
    • System passed, retrieve and record the continuous diagnostic trouble codes (DTCs), erase the continuous DTCs, and carry out self-test diagnostics for the PCM.
  1. If the DTCs retrieved are related to the concern, go to the Passive Anti-Theft System (PATS) — Diagnostic Trouble Codes (DTC) Index .
  1. If no DTCs related to the concern are retrieved, go to Passive Anti-Theft System (PATS) — Symptom Chart .
