Principles of Operation

The all-wheel drive (AWD) system is an electronic shift system that allows the operator to choose between 2 different 4-wheel drive modes. The operator can switch the mode select switch (MSS) between AWD and 4H mode at speeds up to 88 km/h (55 mph).

In AWD mode at rest under cruising conditions, the 4WD control module (PCM) activates the transfer case clutch at a minimum level. This allows for the small difference between the front and rear driveshaft speeds that occur normally when negotiating corners on dry pavement.

When wheel slippage occurs, the 4WD control module (PCM) will increase the torque sent to the transfer case clutch to the front wheels in order to prevent or control wheel slip.

In 4H mode with throttle applied, the 4WD control module (PCM) signals the transfer case clutch to lock the front and rear driveshafts for maximum 4-wheel drive traction. The 4H mode is not intended for use when driving on dry or barely wet pavement. It should be used only for off-road or in severe winter driving conditions.

    Feature Inputs:
  • Mode select switch (MSS)
  • Wheel speeds (via CAN communication)
    Feature Outputs:
  • Transfer case clutch
  • Instrument cluster indicators (via CAN communication)
  • AWD mode (via CAN communication to PCM)

Pinpoint Test S: No Communication With the 4WD Control Module (PCM)

Refer to Wiring Diagrams Cell 34 for schematic and connector information.

Normal Operation

In order for the 4WD Control Module (PCM) to communicate with the diagnostic tool, the operating voltage required to supply the 4WD control module (PCM) is in a range between 9 and 16 volts. Hot-at-all times voltage is supplied by circuit 1861 (RD) and circuit 1865 (WH/OG).

    Possible Causes
  • Power supply:
    • circuit 1861 (RD)
    • circuit 1865 (WH/OG)
  • Central junction box (CJB) mini-fuses:
    • 29 (15A)
    • 30 (15A)
  • Charging system
  • Battery

Pinpoint Test T: The Vehicle Does Not Shift Between AWD And 4H Correctly

Normal Operation

With the key in the ON position, the 4WD control module (PCM) receives the operator's choice of 4-wheel drive selection from the MSS. The 4WD control module (PCM) then controls the electric clutch. If the transfer case is not responding to the operator's intentions, systematically check all inputs and outputs to the 4WD control module (PCM).

    Possible Causes
  • 4H indicator
  • Transfer case
  • 4WD control module (PCM)
  • Circuits:
    • 1856 (GY/OG)
    • 92 (LB/YE)
    • 779 (BN)
    • 682 (DB)
    • 780 (DB)
  • Transfer case clutch
  • Ignition switch and/or circuitry
  • CAN communication network
  • Instrument cluster (IC)
  • Mode select switch (MSS)

Pinpoint Test U: The Instrument Cluster 4H Indicator Does Not Operate Correctly/Does Not Operate

Normal Operation

The 4H indicator status is transmitted to the instrument cluster from the 4WD control module (PCM) through the controller area network (CAN), circuits 1827 (WH/LG) and 1828 (PK/LG). A steady indicator displays when the vehicle is in 4H, according to the driver selection. Depending on the type of failure, the indicator flashes at 2 minute intervals to indicate an 4WD system fault condition. For information about CAN, refer to Section 418-00 .

    Possible Causes
  • Instrument cluster
  • 4WD system fault
  • CAN circuits:
    • 1827 (WH/LG)
    • 1828 (PK/LG)
  • 4WD control module (PCM)

Pinpoint Test V: Straightline Driveline Wind-up

Normal Operation

In order for the 4WD system to function correctly, tires and wheels must be the same size, be in good condition, and the front and rear axle ratios must match.

    Possible Causes
  • Unmatched tire sizes
  • Unequal amounts of tire wear
  • Unequal tire inflation pressures
  • Unmatched front and rear axle ratios

Pinpoint Test W: The Vehicle Binds In Turns/Resists Turning/Pulsates/Shudders In A Straightline

Normal Operation

When the vehicle is operating in 4H, all 4 wheels rotate at the same speed. The amount of traction the vehicle can achieve is dependent on the amount of traction each tire can establish with the road surface. In order for the 4WD system to function correctly, each tire must be in good condition and the front and rear axle ratios must match.

    Possible Causes
  • 4H indicator
  • 4WD control module (PCM)
  • Transfer case and related components
  • Wheel/tire assemblies

Pinpoint Test X: AWD System Functional Test

Normal Operation

When the vehicle operating in 4H mode, all 4 wheels rotate at the same speed. The amount of traction the vehicle can achieve depends on the amount of traction each tire can establish with the road surface. In order for the AWD system to operate correctly, each wheel/tire assembly must have matching sizes, and the front and rear axle ratios must match.

    Possible Causes
  • 4H indicator
  • 4WD control module (PCM)
  • Transfer case and related components
  • Wheel/tire assemblies
