SECTION 303-08: Engine Emission Control
2006 F-150/Mark LT Workshop Manual
REMOVAL AND INSTALLATION
Procedure revision date: 06/29/2005

Exhaust Gas Recirculation (EGR) System Module — 4.2L

Removal

  1. Disconnect the battery ground cable. For additional information, refer to Section 414-01 .
  1. Disconnect the exhaust gas recirculation (EGR) system module vacuum connector.
  1. Disconnect the EGR system module electrical connector.
  1. Disconnect the exhaust manifold-to-EGR system module tube from the EGR system module.
  1. Remove the 2 bolts, the EGR system module and the gasket.
    • Discard the EGR system module gasket.

Installation

  1.  CAUTION: Do not use metal scrapers, wire brushes, power abrasive discs or other abrasive means to clean the sealing surfaces. These tools cause scratches and gouges which make leak paths. Use a plastic scraping tool to remove all traces of the old gasket.

    Clean the EGR system module gasket mating surfaces.
  1. Install a new gasket with the raised circle facing the intake manifold.
  1. Position the EGR system module and install the bolts.
    • Tighten to 25 Nm (18 lb-ft).
  1. Connect the exhaust manifold-to-EGR system module tube to the EGR system module
    • Tighten to 40 Nm (30 lb-ft).
  1. Connect the EGR system module electrical connector.
  1. Connect the EGR system module vacuum connector.
  1. Connect the battery ground cable. For additional information, refer to Section 414-01 .
