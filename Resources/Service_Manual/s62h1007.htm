Principles of Operation

The headlamp switch routes voltage to the parking lamps when demanded.

 

Inspection and Verification

  1. Verify the customer concern.
  1. Visually inspect the following for obvious signs of mechanical or electrical damage.


Visual Inspection Chart
Mechanical Electrical
  • Headlamp switch
  • Central junction box (CJB) fuse 6 (15A) (parking lamps)
  • Bulb(s)
  • Circuitry
  • Vehicle security module

  1. If an obvious cause for an observed or reported concern is found, correct the cause (if possible) before proceeding to the next step.
  1. If the concern is not visually evident, verify the symptom. GO to Symptom Chart .

Pinpoint Tests

 

Pinpoint Test O: The Autolamps Operate, But Parking, Rear or License Plate Lamps Are Inoperative

Normal Operation

The vehicle security module is supplied battery voltage through circuit 195 (TN/WH). When exterior lamps are turned on by the vehicle security module, voltage is routed from circuit 195 (TN/WH) to circuit 14 (BN) through an internal relay.

    Possible Causes
  • Fuse
  • Circuit 14 (BN) open
  • Circuit 195 (TN/WH) open
  • Vehicle security module
 
PINPOINT TEST O: THE AUTOLAMPS OPERATE, BUT PARKING, REAR OR LICENSE PLATE LAMPS ARE INOPERATIVE
Test Step Result / Action to Take
  O1 CHECK THE PARKING LAMP OPERATION  
  • Key in OFF position.
  • Place the headlamp switch in the PARKING LAMPS ON position.
  • Are the parking lamps illuminated?
Yes
TURN the headlamp switch off. GO to O2 .

No
Go To Pinpoint Test P .
  O2 CHECK CIRCUIT 14 (BN) FOR AN OPEN  
  • Disconnect: Vehicle Security Module C3008a.
  • Place the headlamp switch in the PARKING LAMPS ON position.
  • Measure the voltage between the vehicle security module C3008a-6, circuit 14 (BN), harness side and ground

  • Is the voltage greater than 10 volts?
Yes
TURN the headlamp switch off. GO to O3 .

No
REPAIR the circuit. TEST the system for normal operation.
  O3 CHECK CIRCUIT 195 (TN/WH) FOR VOLTAGE  
  • Measure the voltage between the vehicle security module C3008a-7, circuit 195 (TN/WH), harness side and ground.

  • Is the voltage greater than 10 volts?
Yes
GO to O4 .

No
REPAIR the circuit. TEST the system for normal operation.
  O4 CHECK FOR CORRECT VEHICLE SECURITY MODULE OPERATION  
  • Disconnect all the vehicle security module connectors.
  • Check for:
    • corrosion
    • pushed-out pins
  • Connect all the vehicle security module connectors and make sure they seat correctly.
  • Operate the system and verify the concern is still present.
  • Is the concern still present?
Yes
INSTALL a new vehicle security module. REFER to Section 419-10 . TEST the system for normal operation.

No
The system is operating correctly at this time. The concern may have been caused by a loose or corroded connector. CLEAR the DTCs. TEST the system for normal operation.
 

Pinpoint Test P: One Or More Parking, Rear or License Plate Lamps Is Inoperative

Normal Operation

The headlamp switch is supplied battery voltage through circuit 195 (TN/WH). When the headlamp switch is turned to the PARKING LAMPS ON position, the headlamp switch routes voltage to circuit 14 (BN), which then supplies voltage to the exterior lamps. Ground is provided to the bulbs through circuit 57 (BK).

    Possible Causes
  • Fuse
  • Circuit 14 (BN) open
  • Circuit 57 (BK) open
  • Circuit 195 (TN/WH) open
  • Headlamp switch
  • Central junction box (CJB)
 
PINPOINT TEST P: ONE OR MORE PARKING, REAR OR LICENSE PLATE LAMPS IS INOPERATIVE
Test Step Result / Action to Take
  P1 DETERMINE THE CONDITION  
  • Key in OFF position.
  • Place the headlamp switch in the PARKING LAMPS ON position.
  • Are all the exterior lamps inoperative?
Yes
TURN the headlamp switch off. GO to P2 .

No
TURN the headlamp switch off. GO to P5 .
  P2 CHECK THE VOLTAGE TO THE HEADLAMP SWITCH  
  • Disconnect: Headlamp Switch C2150a.
  • Measure the voltage between the headlamp switch C2150a-10, circuit 195 (TN/WH), harness side and ground.

  • Is the voltage greater than 10 volts?
Yes
GO to P4 .

No
VERIFY the CJB fuse 6 (15A) is OK. If OK, GO to P3 .
  P3 CHECK CIRCUIT 195 (TN/WH) FOR AN OPEN  
  • Disconnect: CJB C270f.
  • Measure the resistance between the CJB C270f-4, circuit 195 (TN/WH), harness side and the headlamp switch C2150a-10, circuit 195 (TN/WH), harness side.

  • Is the resistance less than 5 ohms?
Yes
REPAIR or INSTALL a new CJB. TEST the system for normal operation.

No
REPAIR the circuit. TEST the system for normal operation.
  P4 CHECK CIRCUIT 14 (BN) FOR AN OPEN  
  • Connect a fused (15A) jumper wire between the headlamp switch C2150a-10, circuit 195 (TN/WH), harness side and the headlamp switch C2150a-5, circuit 14 (BN), harness side.

  • Do the exterior lamps illuminate?
Yes
INSTALL a new headlamp switch. REFER to Headlamp Switch in this section. TEST the system for normal operation.

No
REPAIR the circuit. TEST the system for normal operation.
  P5 CHECK FOR VOLTAGE TO THE INOPERATIVE LAMP  
  • Disconnect: Inoperative Lamp.
  • Place the headlamp switch in the PARKING LAMPS ON position.
  • Measure the voltage between the inoperative lamp, harness side and ground as follows:


    • Inoperative Lamp Connector-Pin Circuit
    LH front C1023-2 14 (BN)
    LH front side marker C1127-1 14 (BN)
    RH front C1043-2 14 (BN)
    RH front side marker C1126-1 14 (BN)
    LH rear C4035-2 14 (BN)
    RH rear C4032-2 14 (BN)
    LH license plate C452-1 14 (BN)
    RH license plate C462-1 14 (BN)


  • Is the voltage greater than 10 volts?
Yes
REPAIR circuit 57 (BK). TEST the system for normal operation.

No
REPAIR circuit 14 (BN). TEST the system for normal operation.
 

Pinpoint Test Q: The Parking, Rear Or License Plate Lamps Are On Continuously

Normal Operation

The headlamp switch is supplied battery voltage through circuit 195 (TN/WH). When the headlamp switch is turned to the PARKING LAMPS ON position, the headlamp switch supplies voltage to circuit 14 (BN), which then supplies voltage to the exterior lamps.

    Possible Causes
  • Circuit 14 (BN) short to voltage
  • Headlamp switch
  • Vehicle security module
 
PINPOINT TEST Q: THE PARKING, REAR OR LICENSE PLATE LAMPS ARE ON CONTINUOUSLY
Test Step Result / Action to Take
  Q1 CHECK THE HEADLAMP SWITCH  
  • Disconnect: Headlamp Switch C2150a.
  • Do the parking lamps continue to illuminate?
Yes
If equipped with autolamps, GO to Q2 .

If not equipped with autolamps, REPAIR circuit 14 (BN). TEST the system for normal operation.

No
INSTALL a new headlamp switch. REFER to Headlamp Switch in this section. TEST the system for normal operation.
  Q2 CHECK CIRCUIT 14 (BN) FOR SHORT TO POWER  
  • Disconnect: Vehicle Security Module C3008a.
  • Do the parking lamps continue to illuminate?
Yes
REPAIR the circuit. TEST the system for normal operation.

No
GO to Q3 .
  Q3 CHECK FOR CORRECT VEHICLE SECURITY MODULE OPERATION  
  • Disconnect all the vehicle security module connectors.
  • Check for:
    • corrosion
    • pushed-out pins
  • Connect all the vehicle security module connectors and make sure they seat correctly.
  • Operate the system and verify the concern is still present.
  • Is the concern still present?
Yes
INSTALL a new vehicle security module. REFER to Section 419-10 . TEST the system for normal operation.

No
The system is operating correctly at this time. The concern may have been caused by a loose or corroded connector.
