2006 F-150/Mark LT Workshop Manual
DESCRIPTION AND OPERATION
Procedure revision date: 06/29/2005

Steering Column Switches

The steering column switches receive driver inputs and send signals to various components. The multi-function switch controls the windshield wipers, hazard flasher, turn signal, headlamp low/high beam and headlamp dimmer/flash to pass. The ignition switch is used in conjunction with the lock cylinder key to start the vehicle. When the lock cylinder key is switched to different positions it moves linkages, which enable the ignition switch to send battery voltage to various components.
