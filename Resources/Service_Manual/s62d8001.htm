Information And Message Center

Message Center

The message center display is a green vacuum fluorescent, fixed format display, and is controlled by the message center switches. The message center displays important vehicle information by constantly monitoring different vehicle systems. The message center displays a warning message pertaining to the system in which a fault has been detected. Refer to the Owner's Literature for operating instructions.
