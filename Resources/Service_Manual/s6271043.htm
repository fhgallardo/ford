SECTION 307-01: Automatic Transmission — 4R70E/4R75E
2006 F-150/Mark LT Workshop Manual
IN-VEHICLE REPAIR
Procedure revision date: 01/25/2006

Manual Control Lever Shaft and Seal


Special Tool(s)
Alignment Gauge, TR Sensor
307-351 (T97L-70010-A)
Installer, Shift Shaft Fluid Seal
307-050 (T74P-77498-A)

Removal

 CAUTION: Use care not to damage the manual control lever shaft bore. The new seal can leak.

 CAUTION: Discard the outer manual control lever nut.

NOTE: When removing the manual control lever shaft retaining pin, use a soft cloth to protect the surface of the case.

All vehicles

  1. With the vehicle in NEUTRAL, position it on a hoist. For additional information, refer to Section 100-02 .

Four wheel drive (4WD) vehicles

  1. Remove the front driveshaft shield nuts and remove the shield.

  1. NOTE: Index-mark the front and rear flange.

    NOTE: It is not necessary to remove the front driveshaft from the vehicle.

    Index-mark the front and rear flange, disconnect the front driveshaft and position aside.

All vehicles

  1. Loosen, but do not remove the transmission fluid pan bolts and allow the fluid to drain. Once the fluid has drained, remove the bolts.
  1. Remove the transmission fluid pan and gasket.
  1. Disconnect the selector lever cable end.
  1. Disconnect the digital transmission range (TR) electrical connector.

  1.  CAUTION: Discard the outer manual control lever shaft nut. Do not reuse. The old nut will not retain the torque specification.

    Remove the manual control lever.
    1. Remove and discard the manual control lever nut.
    1. Remove the manual control lever.
  1. Remove the digital TR sensor bolts and the sensor.
  1. Remove the manual control valve detent lever bolt and spring.
  1. Remove the manual control lever shaft retaining pin.
  1. Remove the inner manual control lever shaft nut and slide the manual control lever shaft out of the case.
  1. Remove the parking lever actuating rod.
    1. Remove the manual valve detent lever.
    1. Remove the parking lever actuating rod.
  1. Remove the manual control lever shaft seal.

Installation

All vehicles

  1. Using the special tool, install the manual control lever seal.
  1. Install the parking lever actuating rod.
    1. Install the parking lever actuating rod.
    1. Install the manual valve detent lever.
  1. Install the manual control lever shaft.
    1. Install the manual control lever shaft.
    1. Install the inner manual control lever shaft nut.
      • Tighten to 31 Nm (23 lb-ft).
    1. Install the manual valve control lever shaft retaining pin.
  1. Install the manual valve detent lever spring and bolt.
    • Tighten to 10 Nm (89 lb-in).
  1. Install the digital TR sensor and loosely install the bolts.

  1. NOTE: The manual lever must be in the NEUTRAL position.

    Using the special tool, align the digital TR sensor slots. The tool is designed to fit snug.

  1.  CAUTION: Tightening one screw before tightening the other may cause the sensor to bind or become damaged.

    Tighten the digital TR sensor bolts.
    • Tighten to 9 Nm (80 lb-in).
  1. Install the manual control lever.
    1. Position the manual control lever.
    1. Install a new manual control lever nut.
      • Tighten to 33 Nm (24 lb-ft).
  1. Connect the digital TR sensor electrical connector.

  1. NOTE: When installing the selector lever cable, make sure that the selector lever cable locking tabs are locked in place and the cable end is snapped onto the ball stud. Press the selector lever cable into the bracket and listen for the cable to click in place. Pull back on the selector lever cable to make sure that it is locked into the bracket. Also make sure that the selector lever cable end is correctly installed onto the ball stud. Pull back on the selector lever cable end to make sure that the cable end is correctly installed.

    Connect the selector lever cable end to the manual lever with the lever in the (D) position.
  1. Position the pan magnet into the transmission fluid pan.

  1. NOTE: The transmission fluid pan gasket is reusable. Clean and inspect for damage; if not damaged, the gasket should be reused.

    Install the transmission fluid pan and gasket.
    1. Position the transmission fluid pan and gasket.
    1. Install the transmission fluid pan bolts.
      • Tighten to 14 Nm (10 lb-ft).

4WD vehicles

  1. Install the front driveshaft and align the index marks made during removal. Install the 4 bolts.
    • Tighten to 103 Nm (76 lb-ft).
  1. Install the front driveshaft shield and install the 2 nuts.
    • Tighten to 18 Nm (13 lb-ft).

All vehicles

  1. NOTE: Start by filling the transmission with 4.7 liters (5 quarts) of clean automatic transmission fluid.

    Fill the transmission to the correct level with clean automatic transmission fluid.
