Principles of Operation

The vehicle security module controls the power door lock system. The module monitors the door lock/unlock switch inputs from the interior door lock switches. When the module receives an unlock/lock command from any door lock control switch, the module activates the door lock actuators. The module also controls the following remote lock/unlock features:

  • remote keyless entry (RKE) transmitter lock/unlock
  • panic alarm
  • RKE keypad

For additional information, refer to Section 501-14 .

 

Inspection and Verification

  1. Verify the customer concern.
  1. Visually inspect for obvious signs of electrical damage.


Visual Inspection Chart
Electrical
  • Central junction box (CJB) fuses:
    • 18 (10A)
    • 107 (30A)
  • Circuitry
  • Vehicle security module

  1. If an obvious cause for an observed or reported concern is found, correct the cause (if possible) before proceeding to the next step.
  1. If the cause is not visually evident, connect the diagnostic tool to the data link connector (DLC) and select the vehicle to be tested from the diagnostic tool menu. If the diagnostic tool does not communicate with the vehicle:
  • check that the program card is correctly installed.
  • check the connections to the vehicle.
  • check the ignition switch position.
  • check that the locks are in the unlocked position.
  1. If the diagnostic tool still does not communicate with the vehicle, refer to the diagnostic tool operating manual.
  1. Carry out the diagnostic tool data link test. If the diagnostic tool responds with:
  • CAN, ISO or UBP circuit fault; all electronic control units no response/not equipped, refer to Section 418-00 .
  • No response/not equipped for the vehicle security module, Go To Pinpoint Test B .
  • System passed, retrieve and record the continuous diagnostic trouble codes (DTCs), erase the continuous DTCs and carry out self-test diagnostics for the vehicle security module.
  1. If the DTCs retrieved are related to the concern, go to the Diagnostic Trouble Code (DTC) Chart in this section.
  1. If no DTCs related to the concern are retrieved, GO to Symptom Chart .

Pinpoint Tests

 

Pinpoint Test B: No Communication With The Vehicle Security Module

Normal Operation

The vehicle security module receives voltage from the central junction box (CJB) through circuits 7 (LG/YE) and 294 (WH/LB). The vehicle security module is grounded through circuits 57 (BK) and 938 (BK/LG).

    Possible Causes
  • fuse(s)
  • circuit 7 (LG/YE) open
  • circuit 57 (BK) open
  • circuit 294 (WH/LB) open
  • circuit 938 (BK/LG) open
  • vehicle security module
 
PINPOINT TEST B: NO COMMUNICATION WITH THE VEHICLE SECURITY MODULE

 CAUTION: Use the correct probe adapter(s) when making measurements. Failure to use the correct probe adapter(s) may damage the connector.

Test Step Result / Action to Take
  B1 CHECK THE VEHICLE SECURITY MODULE C3008a AND C3008b FOR DAMAGE  
  • Key in OFF position.
  • Disconnect: Vehicle Security Module C3008a and C3008b.
  • Inspect the vehicle security module connectors for damage.
  • Are the vehicle security module connectors OK?
Yes
GO to B2 .

No
REPAIR the vehicle security module connector. CARRY OUT the diagnostic tool data link test.
  B2 CHECK CIRCUITS 7 (LG) AND 294 (WH/LB) FOR VOLTAGE  
  • Key in ON position.
  • Measure the voltage between the vehicle security module C3008a-8, circuit 7 (LG/YE), harness side and ground; and between the vehicle security module C3008b-16, circuit 294 (WH/LB), harness side and ground.

  • Are the voltages greater than 10 volts?
Yes
GO to B3 .

No
VERIFY the CJB fuses 18 (10A) and 107 (30A) are OK. If OK, REPAIR the circuit. CLEAR the diagnostic trouble codes (DTCs). REPEAT the self-test.
  B3 CHECK CIRCUITS 57 (BK) AND 938 (BK/LG) FOR AN OPEN  
  • Key in OFF position.
  • Measure the resistance between the vehicle security module C3008a-4, circuit 57 (BK), harness side and ground; and between the vehicle security module C3008b-2, circuit 938 (BK/LG), harness side and ground.

  • Are the resistances less than 5 ohms?
Yes
REFER to Section 418-00 to continue diagnosis of the communications network.

No
REPAIR the circuit. CLEAR the DTCs. REPEAT the self-test.
