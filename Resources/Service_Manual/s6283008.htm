SECTION 308-03: Manual Transaxle/Transmission
2006 F-150/Mark LT Workshop Manual
DISASSEMBLY AND ASSEMBLY OF SUBASSEMBLIES
Procedure revision date: 02/17/2006

Input Shaft Bearing


Special Tool(s)
Remover, Bearing
205-055 (T71P-4621-B)
Adapter Set, Step Plate
205-DS011 (D80L-630-A)

Disassembly

  1. Remove and discard the fluid scoop ring.

  1.  CAUTION: Use of essential tools may cause damage to the bearings being removed.

     CAUTION: Hold the input shaft so it does not fall.

    Using the special tool, press the input shaft bearing off the input shaft. Discard the bearing.
    • Install a new bearing and cup. Install as a set only. Do not install one without the other.
    • Inspect the input shaft for wear or damage. Install a new gear as necessary.

Assembly

  1.  CAUTION: Press the input shaft bearing onto the shaft by the inner race only. Pressing on the outer race will damage the bearing.

    Using the special tools, press the new input shaft bearing onto the input shaft.
  1. Install the fluid scoop ring.
    • A click should be felt as the fluid scoop ring notches align with the input shaft oil holes.
