SECTION 308-01: Clutch
2006 F-150/Mark LT Workshop Manual
REMOVAL AND INSTALLATION
Procedure revision date: 05/01/2006

Pilot Bearing


Special Tool(s)
Pilot Bearing Replacer
308-105 (T85T-7137-A)
Puller
308-001 (T58L-101-B)

Removal

  1. Using the special tool, remove the input shaft pilot bearing.

Installation

  1. NOTE: The input shaft pilot bearing can only be installed with the seal facing the transmission. The input shaft pilot bearing is pregreased and does not require additional lubrication.

    Using the special tool, install a new input shaft pilot bearing.
