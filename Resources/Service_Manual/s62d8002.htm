Principles of Operation

The message center is an integral part of the instrument cluster. The message center communicates with the powertrain control module (PCM) and the anti-lock brake system (ABS) module over the communication network. It is important to understand the following information.

The 3 message center functions are:

  • information menu
  • setup menu
  • warning messages

The message center information can be selected by pressing any of the following 3 buttons:

  • INFO
  • RESET
  • SETUP

The setup display time-outs and defaults to the fuel blank display function. The temporary alert display interrupts the current display to show the status of an event that has just happened. The warning display interrupts the current display until cleared or reset by the driver.

Information Displays

The information displays are non-timed modes. The selected mode remains on until the driver presses a message center button to change the mode, or it is overridden by another mode. The information display modes are:

  • odometer
  • trip odometer
  • compass
  • outside air temperature
  • distance to empty
  • average fuel economy
  • elapsed trip time
Setup Displays

The setup displays are timed modes and default back to the fuel blank display function. The setup display modes are:

  • English/metric units
  • autolocks on, autolocks off
  • language (English, French or Spanish)
Warning Messages

The warning messages can interrupt the display until cleared or reset by the driver. The system uses 3 types of warning messages: single cycle, nonrenewable, and repetitive. The single cycle warning messages are:

  • DOOR AJAR
  • CHECK FUEL CAP
  • TRANSMISSION

The single cycle of warning messages display once whenever the ignition switch is turned to the ON position or a fault occurs in a system. They can be cleared by pressing the RESET button.

The nonrenewable warning messages are:

  • CHECK ENGINE TEMPERATURE
  • LOW FUEL LEVEL
  • CHECK CHARGING SYSTEM
  • LOW BRAKE FLUID
  • LOW OIL PRESSURE

These warning messages are displayed at a fixed interval whenever the ignition switch is turned to the ON position or is in the ACC position. To remove a repetitive warning message, press the RESET button. The message only recurs after 10 minutes or when the warning condition is corrected.

The third type of warning messages display whenever the ignition switch is turned to the ON position and a fault occurs in a system. The fault must be corrected to clear the repetitive warning messages.

The repetitive warning messages are:

  • REDUCED ENGINE POWER
  • STOP ENGINE SAFELY
 

Inspection and Verification

  1. Verify the customer concern.
  1. Visually inspect for obvious signs of mechanical or electrical damage.


Visual Inspection Chart
Mechanical Electrical
  • Engine oil level
  • Door adjustment
  • Engine coolant level
  • Fuel tank
  • Central junction box (CJB) fuse 18 (10A)
  • Miniature bulb(s)
  • Circuitry
  • Message center switch
  • Instrument cluster

  1. If an obvious cause for an observed or reported concern is found, correct the cause (if possible) before proceeding to the next step.
  1. If the concern is not visually evident, connect the diagnostic tool to the data link connector (DLC) and select the vehicle to be tested from the diagnostic tool menu. If the diagnostic tool does not communicate with the vehicle:
    • check that the program card is correctly installed.
    • check the connections to the vehicle.
    • check the ignition switch position.
  1. If the diagnostic tool still does not communicate with the vehicle, refer to the diagnostic tool operating manual.
  1. Carry out the diagnostic tool data link test. If the diagnostic tool responds with:
    • CAN, UBP or ISO circuit fault; all electronic control units no response/not equipped, refer to Section 418-00 .
    • No response/not equipped for the instrument cluster, refer to Section 413-01 .
    • System passed, retrieve and record the continuous diagnostic trouble codes (DTCs), erase the continuous DTCs, and carry out self-test diagnostics for the instrument cluster.
  1. If the DTCs retrieved are related to the concern, go to the Instrument Cluster Diagnostic Trouble Code (DTC) Index. For all other DTCs, refer to Section 419-10 .
  1. If no DTCs related to the concern are retrieved, GO to Symptom Chart .

Pinpoint Tests

 

Pinpoint Test A: The Message Center Is Inoperative

Normal Operation

The message center display is integrated into the lower RH corner of the instrument cluster. The instrument cluster odometer, trip odometer, compass, outside air temperature and message center displays are contained in the same window. The message center switch is hardwired to the instrument cluster through circuits 1411 (GY/OG) and 1396 (VT/WH). The message center switches use a different resistance value for each switch, allowing the instrument cluster to determine which switch is pressed.

    Possible Causes
  • instrument cluster
  • message center switch
 
PINPOINT TEST A: THE MESSAGE CENTER IS INOPERATIVE
Test Step Result / Action to Take
  A1 CHECK THE INSTRUMENT CLUSTER ODOMETER OPERATION  
  • Key in ON position.
  • Observe the odometer display in the instrument cluster.
  • Is the odometer displayed?
Yes
The system is OK. If the INFO, SETUP or RESET buttons are inoperative, Go To Pinpoint Test F .

No
REFER to Section 413-01 to continue diagnosis of the odometer.
 

Pinpoint Test B: The Compass Is Inoperative

Normal Operation

The compass sensor module receives voltage from the central junction box (CJB) through circuit 294 (WH/LB) and is grounded through circuit 875 (BK/LB). The compass sensor module communicates vehicle direction to the instrument cluster to be displayed in the message center. The compass sensor module uses circuit 702 (WH/BK) and circuit 703 (WH/OG) to communicate to the instrument cluster.

    Possible Causes
  • fuse
  • circuit 294 (WH/LB) open
  • circuit 702 (WH/BK) open or short to ground
  • circuit 703 (WH/OG) open or short to ground
  • circuit 875 (BK/LB) open
  • compass sensor module
  • instrument cluster

Pinpoint Test D: The Transmission Display is Inoperative

Normal Operation

The instrument cluster and message center receive transmission data from the powertrain control module (PCM) over the communication network. When a concern exists with the transmission, the PCM provides a command signal to the message center to display the transmission warning message.

    Possible Causes
  • instrument cluster

Pinpoint Test E: The Check Fuel Cap Display Is Inoperative

Normal Operation

The powertrain control module (PCM) monitors the fuel tank evaporative system for significant leaks that occur following the refueling of the vehicle. If a fuel tank pressure change greater than a -23.7 kPa (-7 in Hg) of vacuum occurs within 30 seconds after refueling, or there is excessive purge (fuel vapor) flow greater than 454 g (1.0 lb) per minute, the PCM sets DTC P0457 and sends a message to the instrument cluster over the communication network to turn on the check fuel cap display in the message center.

    Possible Causes
  • instrument cluster

Pinpoint Test F: The Message Center Switch Does Not Operate Correctly

Normal Operation

The message center switch communicates the requested switch function to the instrument cluster through circuits 1411 (GY/OG) and 1396 (VT/WH).

    Possible Causes
  • circuit 1396 (VT/WH) open or short to ground
  • circuit 1411 (GY/OG) open or short to ground
  • instrument cluster
  • message center switch
