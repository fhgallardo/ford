SECTION 307-01: Automatic Transmission — 4R70E/4R75E
2006 F-150/Mark LT Workshop Manual
IN-VEHICLE REPAIR
Procedure revision date: 06/29/2005

2-3 Accumulator

Removal

  1. Remove the main control valve body. For additional information, refer to Main Control Valve Body in this section.
  1. Remove the 2-3 accumulator piston retainer.
  1. Remove the accumulator piston and spring.

Installation

  1. Install the 2-3 accumulator assembly.
    1. Install the accumulator piston.
    1. Install the accumulator piston spring.
    1. Install the accumulator spring retainer.
  1. Install the main control valve body. For additional information, refer to Main Control Valve Body in this section.
