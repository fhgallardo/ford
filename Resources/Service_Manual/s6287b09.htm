Disassembly

  1.  WARNING: Wear protective eyewear whenever using compressed air.

    Clean the transfer case with solvent and dry with compressed air.

  1.  WARNING: Make sure the special tool lock pin is secure.

    Using the special tool, secure the transfer case to the bench.
  1. If not done previously, remove the drain plug and drain the fluid.
    • DRAIN fluid and INSPECT for contamination. If heavy metallic or plastic contamination is noted, REMOVE transfer case and INSPECT for source of contamination. INSTALL a new components as necessary.
  1. Remove the coil wire electrical connector.
    • Remove the wire connector spacer.
    • Using an electrical connector pin extractor tool, remove the coil wire and pin.
  1. Install a 102 mm (4.0 in) piece of 31.75 mm (1.25 in) i.d. conduit on the output shaft into the extension housing to protect it from damage and keep the bushing from separating from the housing. Remove the 6 extension housing bolts.
  1. Using a soft faced hammer, tap the extension housing to break the gasket seal and remove the extension housing.
    • If the extension housing bushing is dislodged during removal, a new extension housing bushing must be installed.
  1. Remove the 3 coil-to-cover nuts and the coil.
  1. Remove the rear output shaft snap ring.

  1. NOTE: The bolts are self-tapping, and it is normal to find metal shavings while removing the cover.

    Remove the case bolts.

  1. NOTE: Use pry bosses to separate the cover from the case.

    Using a suitable pry bar, separate the transfer case covers.
  1. Remove the bearing.
  1. Remove the inboard bearing snap ring.
  1. Remove the coil housing and thrust washer assembly.
  1. Remove the 3 bearings and the apply cam.
  1. Remove the wave spring.
  1. Remove the snap ring.
  1. Remove the clutch pack assembly.
  1. Remove the spacer.
  1. Remove the drive chain and sprockets as an assembly.
  1. Remove the magnet from its slot in the case.
  1. Remove the thrust bearing race.
  1. Remove the thrust bearing.
  1. Remove the thrust bearing race.
  1. Remove the output shaft hub.
  1. Remove the oil pump and inlet tube.
  1. Remove the output shaft.
  1. Remove the input gear.
  1. Remove the snap ring.
  1. Remove the input shaft support bearing.
  1. Remove the input seal.
  1. Remove the snap ring and the front output shaft.
  1. Using 2 crow's foot prybars, remove the front output shaft bearing.
  1. Remove the front output shaft seal.
  1. Remove the dust shield, if necessary.

Assembly

  1. Lubricate all internal components with the recommended transfer case lubricant during reassembly.
  1. Clean the mating surfaces of the cases.
  1. Using the special tools, install the input shaft support bearing.
  1. Using the special tools, install the front output shaft bearing.
  1. Using the special tool, install the front yoke-to-flange seal.
  1. Using the special tools, install a new rear output seal.
  1. Install the dust shield, if removed.
  1. Install the front output shaft and snap ring.
  1. Using the special tools, install the input seal.
  1. Install the rear output shaft sleeve and snap ring.
  1. Install the output shaft.
  1. Install the oil pump assembly.
    • Make sure the oil pump finger is in the slot in the case
  1. Install the oil pan magnet into its slot in the case.
  1. Install the output shaft hub.
  1. Install the thrust bearing race.
  1. Install the thrust bearing.
  1. Install the thrust bearing race.
  1. Install the drive chain and the sprockets as an assembly.
  1. Install the spacer.
  1. Install the clutch pack assembly.
  1. Install the snap ring.
  1. Install the wave spring.
  1. Install the bearings and the apply cam.
  1. Install the cam/coil housing.
  1. Install the snap ring.
  1. Using a suitable press and the special tool, install the rear output shaft support bearing.
  1. Using a suitable press and the special tool, install the front output shaft rear bearing.
  1. Using the special tool, install the rear output shaft bearing.
  1. Using the special tool, install the front output shaft bearing.
  1. Install the coil.
    1. Position the coil.
    1. Install the nuts.
      • Tighten to 10 Nm (89 lb-in).

  1.  CAUTION: If too much silicone rubber is used when sealing the cover to the case, the excess sealant can plug the fluid filter and cause case failure.

    Coat the mating surfaces of the case and cover with a small bead of sealant.
  1. Position the 2 transfer case halves and tighten the bolts according to sequence shown.
    • Tighten to 24 Nm (18 lb-ft).
  1. Install the rear output shaft snap ring.
    • Lift up on the output shaft while installing the snap ring.
  1. Apply a small bead of sealant to the transfer case cover and extension housing mating surfaces.
  1. Install the extension housing.
    1. Position the brown clutch coil wire through its hole in the extension housing.
    1. Position the extension housing.
    1. Install the extension housing bolts.
      • Tighten to 24 Nm (18 lb-ft).
  1. Using the special tool, install the oil seal, if removed.
  1. Install the coil wire into the shift motor electrical connector.
    1. Install the coil wire and pin.
    1. Install the wire connector spacer.
  1. Remove the transfer case from the special tool.
  1. Install the transfer case in the vehicle. For additional information, refer to Transfer Case in this section.
  1. Fill the transfer case. For additional information, refer to Transfer Case Draining and Filling .
