Procedure revision date: 06/29/2005

Disc Check


Special Tool(s)
Dial Indicator/Magnetic Base
100-D002 (D78P-4201-B) or equivalent
  1. Remove clutch assembly. For additional information, refer to Section 308-01 .

  1. NOTE: Use emery cloth to remove minor imperfections in the clutch disc lining surface.

    Inspect clutch disc for:
    • oil or grease saturation.
    • worn or loose rivets at the hub.
    • broken springs.
    • wear or rust on splines.
      Install a new clutch disc if any of these conditions are present.
  1. Use a slide caliper to measure the depth to the rivet heads.
    • Install a new clutch disc if not within specification.
  1. Using a suitable dial indicator, measure the clutch disc runout.
    • Install a new clutch disc if not within specification.
