SECTION 307-01: Automatic Transmission — 4R70E/4R75E
2006 F-150/Mark LT Workshop Manual
IN-VEHICLE REPAIR
Procedure revision date: 06/29/2005

Reverse Servo Assembly


Special Tool(s)
Dial Indicator Gauge with Holding Fixture
100-002 (TOOL-4201-C)
Remover/Installer, Servo Piston
307-251 (T92P-70023-A)
Installer, Servo Piston
307-073 (T80L-77030-A)

Removal

  1. Remove the main control valve body. For additional information, refer to Main Control Valve Body in this section.
  1. Using the special tool, remove the reverse band servo retaining ring.
    1. Compress the servo spring.
    1. Remove the reverse band servo retaining ring.
  1. Remove the reverse servo assembly.
    1. Remove the reverse band servo cover.
    1. Remove the reverse band servo piston and rod.
    1. Remove the reverse band servo spring.

Installation

NOTE: This is not an ordinary installation procedure and does not compensate for band wear.

  1. NOTE: Lubricate the reverse piston seal to facilitate assembly and prevent damage to the seal.

    Install the reverse servo return spring and piston.
    • Do not install the piston cover.
  1. Install the special tool and tighten the band apply bolt.
    • Tighten to 5.6 Nm (50 lb-in).
  1. Attach the special tool to the transmission.
    • Position the indicator stem on the flat portion of the reverse servo piston and zero the dial indicator.
  1. Loosen the bolt until the piston stops against the tool.
  1. Verify that the amount of piston travel on the dial indicator is within specification.
  1. If piston travel is not within specification, select and install the correct servo piston assembly to bring the servo piston travel within specification.
    1. One groove
    1. Two groove
    1. Three groove
  1. Remove the dial indicator and servo selection tool.
  1. Install the correct reverse servo assembly.
    1. Install the reverse band servo spring.
    1. Install the reverse band servo piston and rod.
    1. Install the reverse band servo cover.
  1. Using the special tool, install the reverse servo retaining ring.
    1. Compress the servo spring.
    1. Install the reverse band servo retaining ring.
  1. Install the main control valve body. For additional information, refer to Main Control Valve Body in this section.
